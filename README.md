# Лабораторная работа №5

Реализация собственного механизма внедрения зависимостей (Dependency Injection).

## Технологии и инструменты
* **Java**
* **Java Reflection API**
* **Maven**
* **JUnit 5**
* **Git**

## Структура проекта
* `src/main/java/` — исходный код приложения. Содержит ядро механизма внедрения `Injector`, аннотацию `@AutoInjectable`, тестовые модели (интерфейсы и их реализации) и класс `Main` для демонстрации. Код покрыт JavaDoc комментариями.
* `src/main/resources/` — директория для хранения конфигурационного файла `config.properties` с маппингом зависимостей.
* `src/test/java/` — Unit-тесты, проверяющие корректность работы механизма рефлексии (проверка успешной инициализации полей целевыми экземплярами классов в зависимости от настроек).
